/data/sendSimulData?simulationName&analysisName&partitionId&type

Methods

POST

This method allows data insertion on HBase database given the simulation name and analysis type as parameters

request query parameters
parametervaluedescription

simulationName

string

Type of simulation input data (DEM, FEM, etc)

analysisName

string

Name of the simulation data file

partitionId

string

Identifier of simulation part

type

string

Type of coordinate set (only for DEM)

available response representations:

  • application/xml
